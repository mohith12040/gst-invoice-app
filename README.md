# Lalitha Agencies — GST Invoice Generator

## Professional GST Invoice Management Web Application

A modern and responsive GST Invoice Generator web application built for generating GST purchase invoices instantly with automatic tax calculations and Excel export support.

Designed specifically for:
- GST invoice management
- Financial year–wise invoice tracking
- Automatic SGST/CGST/IGST calculation
- Excel sheet generation
- Business accounting workflows

# ✨ Features

## 📅 Multi Financial Year Support

Supports:
- FY 2026–27
- FY 2027–28
- FY 2028–29

Each financial year contains:
- Dedicated invoice management
- Month selection
- Separate GST calculations
- Financial-year-specific export handling

---

# 🧾 Automatic GST Calculation

The application automatically calculates:

- Taxable Value
- SGST
- CGST
- IGST
- Grand Total

Supported GST rates:
- 5%
- 12%
- 18%
- 28%
- 18% IGST

---
